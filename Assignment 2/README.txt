Camera key bindings:
W,A,S,D:Move ahead,back,left,right respectively

Mouse controls:
Move the mouse to turn the camera in the desired direction
Scroll wheel up/down- Zoom in/out

Note: GLFW/GLEW, Assemp used in the code. Kindly check the PATH directory and make a MakeFile (Location: OpenGLRoot/External Resources/)
OR Run the project in Visual Studio
